[일반user 네비게이션]
┌─────────────────────────────────────────────────┐
│ FASTMATCH    [메인] [제안요청] [제안서] 👤 홍길동│
└─────────────────────────────────────────────────┘
│
├─ 메인           → /
├─ 제안 요청      → /proposals/request

├─ 제안서 목록     → /proposals
└─ 프로필 메뉴
   ├─ 내 정보
   ├─ 내가 등록한 옵션
   └─ 로그아웃

[admin user 네비게이션]
┌──────────────────────────────────────────────────┐
│ FASTMATCH [메인] [관리자] [제안요청] 👤 관리자    │
└──────────────────────────────────────────────────┘
│
├─ 메인          → /
├─ 관리자        → /admin

│  ├─ 매니저 관리  → /admin/managers
│  ├─ 지점 관리    → /admin/branches
│  ├─ 옵션 관리    → /admin/options
│  ├─ 삭제 요청    → /admin/delete-requests
│  └─ 사용자 관리  → /admin/users
├─ 제안 요청     → /proposals/request
├─ 제안서 목록    → /proposals
└─ 프로필 메뉴
   ├─ 내 정보
   └─ 로그아웃

[주요 플로우]
[일반 User]
1. / (메인 페이지)
2. [+ 옵션 등록] 클릭
3. /options/register (옵션 등록)
4. 등록 완료 → / (메인 페이지)
5. 옵션 체크박스 선택 (여러 개)
6. 푸터 [제안서 생성] 버튼 클릭
7. /proposals/create (제안서 생성)
   └─ Step 1: 옵션 순서/정보 변경
   └─ Step 2: 제안서명 작성
   └─ Step 3: PDF 출력

[제안요청 > 추가 > 변경 플로우]
[일반 User]
1. /proposals/request (제안 요청 등록)
2. 브랜드 선택 (패스트파이브, 스파크플러스, 위워크)
3. 이메일 발송 완료
4. /proposals/requests (제안 요청 관리)
5. 특정 제안 클릭 → /proposals/requests/:id (상세)

[추가 제안]
6. [추가 제안 요청] 클릭
7. /proposals/requests/:id/add
8. 새 브랜드 선택 (마이워크스페이스, 헤이그라운드)
9. 추가 이메일 발송 완료

[변경 제안]
6. [변경 제안 요청] 클릭
7. /proposals/requests/:id/modify
8. 조건 수정 (입주일, 인원 등)
9. 기존 브랜드에게 변경 이메일 발송 완료

[옵션 삭제 요청 > 관리자 승인/거부 플로우]
[일반 User]
1. / (메인 페이지)
2. 본인이 등록한 옵션 카드에서 [삭제요청] 클릭
3. 삭제 사유 입력
4. 옵션 상태 → "삭제요청중"

[Admin]
5. /admin/delete-requests (삭제 요청 목록)
6. 특정 요청 클릭
7. /admin/delete-requests/:id/review
8-a. [승인] → 옵션 상태 "삭제됨"
8-b. [거부] → 옵션 상태 "활성" (복구)
